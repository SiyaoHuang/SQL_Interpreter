# cs4321_p1
## Prerequisite
- Maven v.10.0  
- Oracle JDK 1.7+  
## Installation
- Build

    ```
    mvn package
    ```
    The output **.jar** will be built in **/Target**  
- IDE

    We recommend Visual Studio Code as the IDE.  
    If you want to use Eclipse, try the command below:  
    ```
    mvn eclipse:eclipse
    ```
